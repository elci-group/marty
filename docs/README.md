# 🚀 Marty: Your Intelligent File System Navigator 🧭

Welcome to Marty, the smart way to get around your filesystem! 

Marty is a command-line companion that learns your habits to make directory navigation faster and more intuitive. Spend less time `cd`-ing and more time working.

## ✨ Features

*   🔥 **Hotspots:** Marty keeps track of your most frequently visited directories.
*   🧠 **Beliefs:** Create custom links and relationships between directories.
*   📜 **Trace:** A searchable history of your recent navigation.
*   🤖 **Interactive Mode:** A powerful REPL for lightning-fast jumps.
*   🌐 **HTTP Dashboard:** A web UI to visualize your navigation patterns.

## 🛠️ Getting Started

Ready to supercharge your navigation? You'll need Rust installed.

```bash
# Build the application
cargo build --release

# See what's possible
./target/release/marty --help
```
